Proyectos web

NEONWAVE SUSCHI BAR

Este proyecto fue un trabajo colaborativo con un equipo en el que nos desempeñamos en la creación de un sitio web para un restaurante de sushi con una temática Neon. Nos encargamos del diseño y, en mi caso, trabajé principalmente en la "Home". En este proyecto se utilizaron tecnologías como HTML5 y CSS3.

Wirefrime Responsivo (Mobile-first)

Este proyecto sigue la metodología "Mobile-first", donde la página se desarrolla inicialmente para dispositivos móviles y luego se adapta a un sitio web de escritorio, manteniendo un diseño responsivo. En este proyecto, la página actúa como un único wireframe, utilizando HTML5, CSS3 y, en este caso, JavaScript.
